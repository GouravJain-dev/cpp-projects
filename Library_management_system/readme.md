# 📚 Library Management System (C++)

A simple **console-based Library Management System** built using **C++**.
This project allows users to manage books in a library through a menu-driven interface.

---

## 🚀 Features

* Add new books
* View all books
* Search book by ID
* Issue book
* Return book
* Display book status (Available / Issued)
* Unique book ID validation

---

## 🛠 Technologies Used

* C++
* Structures
* Arrays
* Loops and Conditional Statements
* Console-based interface

---

## 📋 Menu Options

1. Add Book
2. View Books
3. Search Book
4. Issue Book
5. Return Book
6. Exit
